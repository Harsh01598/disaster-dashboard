India Disaster Management Dashboard Preview

This is a preview of the disaster management dashboard prototype. The actual dashboard features an interactive map of India with disaster alerts and resource management tools.

View Interactive Dashboard

Dashboard Features

  • Interactive India Map - View state boundaries and active disaster locations
  • Expandable Control Panel - Access all dashboard features from the top-right corner
  • Real-time Alerts - Stay updated with the latest disaster warnings
  • Emergency Contacts - Quick access to important emergency numbers
  • Resource Allocation - Visual representation of deployed resources
  • Responsive Design - Works on desktop and mobile devices
